# 🚀 快速部署指南 - 您的服务器IP: 8.141.102.221

## 📋 完整部署步骤

---

### 第一步：连接到您的服务器

**打开 PowerShell，运行：**
```bash
ssh root@8.141.102.221
```

（输入您的服务器密码后登录）

---

### 第二步：上传部署文件（保持 SSH 连接打开，打开新的 PowerShell）

**在新的 PowerShell 窗口中，进入项目目录：**
```powershell
cd d:\Doc\HBuilderProjects\ChiHai
```

**上传部署文件：**
```powershell
scp d:\Doc\HBuilderProjects\ChiHai\nginx.conf root@8.141.102.221:/tmp/
scp d:\Doc\HBuilderProjects\ChiHai\deploy.sh root@8.141.102.221:/tmp/
```

---

### 第三步：在服务器上运行部署脚本（回到第一个 SSH 连接的 PowerShell 窗口）

```bash
cd /tmp
chmod +x deploy.sh
sudo bash deploy.sh
```

等待脚本运行完成...

---

### 第四步：上传前端和后端文件（还是在刚才的新 PowerShell 窗口）

**上传前端文件：**
```powershell
scp -r d:\Doc\HBuilderProjects\ChiHai\dist\* root@8.141.102.221:/var/www/chihai/
```

**上传后端文件：**
```powershell
scp d:\Doc\HBuilderProjects\ChiHai\server\index.js root@8.141.102.221:/var/www/chihai-backend/server/
```

---

### 第五步：启动后端服务（回到第一个 SSH 连接的 PowerShell 窗口）

```bash
cd /var/www/chihai-backend/server
pm2 start index.js --name chihai-backend
pm2 save
```

---

### 第六步：设置文件权限（继续在服务器 SSH 窗口）

```bash
sudo chown -R www-data:www-data /var/www/chihai
sudo chmod -R 755 /var/www/chihai
sudo chown -R root:root /var/www/chihai-backend
sudo chmod -R 755 /var/www/chihai-backend
```

---

### 第七步：验证部署 🎉

**在浏览器中访问：**
```
http://8.141.102.221
```

**访问管理后台：**
```
http://8.141.102.221/admin
```
